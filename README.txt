Here you can find a Java application intendet for creating and manipulating
with x509 certs.
App is capable of creating a keyPair and exporting it to the x509 cert which is signed by
the application (app simulates a cert authority)

App has a GUI, and this makes it easy to use.

App uses the bouncycastle library, and is capable to encypher the files it creates using a user password.

In this folder you can also find assigent reqirements marked as Asigment
